# Introduction

Common to all HTML elements.

# Attributes

## Most common

- **id**: unique identifier in the document
- **class**: List of the classes of the element.
- **style**: Contains CSS styling
- **title**: Could be used for tooltip information
- **Role and aria-\*:** Used for ensuring accessibility.
- **autofocus**: Is to be focused on page load
- **contenteditable**: indicating if the element should be editable by the user (p, heading, div, span...)
- **draggable**: Whether the element can be dragged (with the drap & drop [API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_Drag_and_Drop_API))
- **enterkeyhint**: The behavior of the "Enter" key
- **hidden**: the element is not yet, or is no longer, relevant. The browser won't render such elements.
- **inert**: Makes the browser disregard user input events for the element.
- **is**: behave like a registered custom built-in element.
- **lang**: Define the language of an element
- **tabindex**: An integer attribute indicating if the element can take input focus

## Others

- **autocapitalize**: Controls whether and how text input is automatically capitalized.
- **data-\***: Custom data attributes
- **accesskey:** Activate or focus on that element using a keyboard combination. (Varies depending on the browser)
- **dir**: The directionality of the element's text (ltr, rtl, auto)
- **inputmode**: the type of virtual keyboard configuration to use when editing this element or its contents.
- **itemid**: Unique, global identifier of an item.
- **itemprop**: Add properties to an item.
- **itemref**: used in conjunction with the itemscope and itemprop attributes to specify additional properties of a microdata item.
- **itemscope**: A boolean to specifiy that the itemprop should be used
- **itemtype**: Specifies the URL of the vocabulary that will be used to define itemprop
- **nonce**: A cryptographic nonce ("number used once")
- **part**: A space-separated list of the part names of the element.
- **slot**: Assigns a slot in a shadow DOM shadow tree.
- **spellcheck**: whether the element may be checked for spelling errors.
- **translate**: specify whether an element's attribute values and the values of its Text are to be translated when the page is localized
- **virtualkeyboardpolicy**: control the on-screen virtual keyboard behavior
